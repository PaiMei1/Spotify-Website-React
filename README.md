# Spotify Website Clone

A Spotify website recreation built with React. Focused on modern UI, responsive layouts, and component-based architecture while practicing frontend development skills.

## Features

- Spotify-inspired UI design
- Reusable React components
- Interactive navigation
- Music/playlist sections
- Modern CSS styling
- Clean project structure
- Performance-focused design
- And much more


## Installation

git clone <repository-url>
cd spotify-website-clone-react
npm install
npm run dev
